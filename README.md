# escooters
# Weather Impact on Shared E-Scooter Usage

This repository documents my contribution to a university group project conducted at the University of Tübingen for Data Literacy.

## Project Overview

The project investigated factors influencing shared e-scooter usage across multiple German cities using mobility and weather data.

## My Contribution

My responsibilities included:

- Collecting weather data using the Open-Meteo API
- Cleaning and preprocessing weather datasets
- Engineering weather-related features (e.g. Wind Chill Index)
- Integrating weather and mobility data
- Conducting statistical analyses (OLS regression, correlation analysis)
- Creating visualizations and interpreting results
